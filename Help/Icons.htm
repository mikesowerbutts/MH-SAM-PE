Icons

Tap to select any of the following icons:

 

o eLearning – launches the MH eLearning courses login page.

 

o Manager’s Notes – launches the Manager’s Notes dialog; scroll down to view all inputs, along with the Manager Review Date.

 

o Additional Notes – launches the Additional Notes dialog.

 

o Print PDF – launches the Print dialog.

 

o Help  – launches the user help file.

 

o Save – saves the Sheet data to the iPad; a message displays when data has successfully saved.

 

o Close – closes the Sheet.
