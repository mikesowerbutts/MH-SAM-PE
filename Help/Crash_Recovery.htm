Crash Recovery

Because the MH PE application uses a significant amount of memory, especially when downloading & uploading data, the application may crash at times.  When this occurs, it is recommended that you close the application from iPad memory so that it opens fresh and with the maximum memory available for the next time.   

 

From the iPad desktop screen, where the icons are displaying, double press the Home button and a list will appear along the bottom.  Press and hold the MH PE icon until it wobbles, then press the "-" to close it from memory.  Press the home button again to close the list.
